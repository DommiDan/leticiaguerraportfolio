Eat-A-Burger

Eat-A-Burger is a full stack development app that tells the user to: 1) Add a Burger Name 2) Choose to Digest the Burger 3) Then tell the user when it's been digested. Even though it is a very simple app. I'm very proud of this assignment, It not only uses Front-End but Back-end elements as well It even focuses on using MySql databases and handlebars. It also taught me about managing a lot of files all at once and file structure.
